Projet : France Mobilier

  • Rôle : Création de projet
  • Client : Professeur de formation
  • Date du projet : 28 Septembre, 2022

But du projet :

Le projet consiste en la création d'un site web en PHP permettant de répertorier des meubles en fonction d'une gamme et d'une catégorie. Les données utilisées seront stockées dans une base de données MySQL gérée via phpMyAdmin. Le site permettra également aux utilisateurs de s'inscrire à une newsletter pour recevoir des mises à jour régulières sur les nouveaux meubles ajoutés au catalogue. Lorsqu'un utilisateur s'inscrit à la newsletter, son adresse e-mail sera stockée dans une base de données et utilisée pour l'envoi de la newsletter. Ce projet permettra aux utilisateurs de trouver facilement les meubles qu'ils recherchent tout en restant informés des nouveautés.
